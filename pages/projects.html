Alarm Clock Project

Project purpose was to create a functional alarm clock utilizing html, css, and javascript. As a fan of syfy my immediate idea revolved around futuristic vs analog clocks. Once the concept was cristilized I proceeded to research futurist digital clocks. The final product was aimed at portarying the feeling on a clock that projected onto surfaces.

The javascript was fairly straight forward. The issue arrose with trying to figure out how to create an option for users to listen to alarm sounds and selected the noise they wanted to heaar. First I needed to assign a value to each option and connect it with the src path of the music files. Then create an object that would play the music on select without an audio tag.   Live Demo

Frogger Animation Project

At Per Scholas we were delving more into using javascript as a tool for creating functional websites. Part of our lesson was animation and how to encorporate that on the web. From that lesson we were given the assignment of using animation to create movement on the web. We were given the craetive freedom to craete whatever we wanted.

Each item in the frogger game is animated by incrimenting their margins from left to right in a set interval. The frog is utilizing the key down method to give the user the ability to move in any direction using the arrow keys.   Live Demo

Matrix Calculator Project

The project scope was to create a functional calculator with javascript. The complication was adding a button that could provide the average of a series of numbers.

As with most of my projects my ideas stemed from futuristic appearance and the matrix. I modeled the calculator after a the buttons in a pay phone dail in order to emulate the scenes in The Matrix where characters must call an operator to escape The Matrix. When the = button is clicked css elements are changed with javascript to turn the container background into a Matrix gif. When the Call button is clicked css elements are changed once again to revert the backgrounds to normal.   Live Demo
